@startuml
actor Analyst
node System
agent DBagent
agent Transformer
database "Vector DB" as vdb
cloud "api" as api
file "Generated Report" as gr

' Relationships
Analyst --> System: interact
System --> DBagent: process request

System <- api: fetch 
DBagent --> vdb: store vectors
DBagent <-- vdb: retrieve vectors

DBagent --> Transformer: provide context
Transformer --> System: return output
System --> gr: generate report

note top of DBagent
  Single DBagent instance handles:
  * Embedding
  * Storage
  * Retrieval
  * Context formatting
end note
@enduml
